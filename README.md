## 功能

- 经典益智交互合集（12款）
- 复古 Game Boy 风格界面
- 十字键 + A/B 按键操控
- 支持手机、平板、电视等设备
- 每款独立计分，记录最佳成绩
- 按键音效与震动反馈

## 包含内容

- 2048、扫雷、贪吃蛇、俄罗斯方块、打砖块
- 五子棋、围棋、井字棋（AI交互）
- 吃豆人、赛车、飞机大战
- 华容道（关卡系统）

## 开发环境

- HarmonyOS SDK 6.0.2 (API 22)
- ArkUI + ArkTS
